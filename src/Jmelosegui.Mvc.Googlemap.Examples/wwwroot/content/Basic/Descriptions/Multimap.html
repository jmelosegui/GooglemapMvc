The following sample show how to render multiple maps in a single view 

     <div class="row">
        <div class="col-md-6" style="height: 300px">
            @(
                Html.GoogleMap()
                    .Name("map")
             )
        </div>
        <div class="col-md-6" style="height: 300px">
            @(
                Html.GoogleMap()
                    .Name("map1")
            )
        </div>
    </div>
